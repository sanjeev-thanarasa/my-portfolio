📺 Admin Panel UI – Dish TV Customer & Payment Management

A professionally designed admin dashboard UI for a Dish TV company to manage their users, payments, and subscriptions. This project was fully designed in Figma with attention to user experience, simplicity, and admin efficiency.

Designed Pages and Features:
  • 🔐 Admin Login & Logout pages
  • 📊 Dashboard with summary of total, active, and expired users
  • 👥 Active Users Page – List with filtering and searching
  • ❌ Expired Users Page – Track overdue subscriptions
  • 🔍 Advanced Search & Filter (by name, contact, date, etc.)
  • 📝 User Management – Create, Edit, Delete entries
  • 💬 Send Reminder Messages to Users
User Details Managed:
  • 🆔 User ID
  • 👤 Name
  • 📡 Dish No
  • 📞 Contact Number
  • 📍 Address
  • 📦 Package Type
  • 📅 Activation Date
  • 🔁 Renew Date

This design was intended to streamline Dish TV service operations, enhance user tracking, and support better subscription management. The UI is ready to be converted into a Flutter Web app or other modern tech stack for future development.
